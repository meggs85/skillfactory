Данные на вход подали достаточно чистые, иногда с пропусками Nan  ~4-10%. 
В модели много категориалтных признаков (хоть и представленных цифрами, но всеже категориальными (дискретными) по сути), из которых значимым для таргета оказалось лишь _занятие матери_ по результатам Т-теста. 
Из вещественнных признаков не обнаружено созависимых, из вещественных на таргет-переменную выявлено влияние  _возраста_ (наиболее выражено) и _количества пропусков_.



→ этапы работы над проектом.
1) замена типов данных для категориальных признаков, устранение ошибок
2) анализ распределений и пропусков, устранение выбросов
3) корреляционный анализ вещественных признаков, поиск лишних.
4) проведение т-теста на взаимосвязь между номинативными и таргет-переменной
5) усечение датасета до минимальной информативной формы для дальнейших исследований
